# AI-Recipe-Generator-v3
AI-Powered Recipe Generator is a chatbot that creates unique recipes for any food item using Rasa, SpaCy, and Kaggle dataset for training model. It understands user queries, extracts ingredients and dietary preferences, and generates customized recipes based on real datasets. Link: https://www.kaggle.com/datasets/shuyangli94/food-com-recipes-and-user-interactions
